## :man_teacher: 프로젝트 소개
Kiloflow는 사용자들이 자신의 건강 목표를 효과적으로 관리하고 달성할 수 있도록 지원하는 포괄적인 건강 관리 플랫폼입니다.
이 웹앱은 일상적인 식단과 운동 활동을 간편하게 기록하고, 이를 통해 개인 맞춤형 건강 목표를 달성할 수 있도록 설계되었습니다.
